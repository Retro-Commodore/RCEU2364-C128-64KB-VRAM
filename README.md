# RCEU2364-C128-64KB-VRAM
 64KB VRAM upgrade to the C128/C128D (Not C128DCR as it already has)
 
 See more colors, get higher resolutions, Watch Risen to Oblivion demo as it should be seen.
 
 NOTE: this revision is yet not testet, but should work.
 
 
# BOM
 Optional  
 2x 18pin DIL Sockets  
 1x 48pin DIL Socket  
 
 Required  
 2x 24pin DIL Male/Male header (preferably turned)  
 22nF ceramic capacitor  
 2x 4464 or 41464 4bit 64K Ram chips  
 
 
# How to place components

Carefully pull out the 80 column graphics chip MOS8563, and insert/solder (highly recommended to use a socket) it into the socket of the vram expansion.  
Insert the vram expansion into the socket where the MOS8563 was sitting.
